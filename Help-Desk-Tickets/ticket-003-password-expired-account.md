# Ticket #003 - Password Expired / Unable to Log In

## Ticket Information

**Ticket ID:** INC0012855  
**Priority:** High  
**User:** Maria Garcia  
**Department:** Engineering  
**Location:** Floor 3

---

## Issue Description

The user returned from a 3-week vacation and was unable to log into her computer. She received a message stating that her password had expired and needed to be changed. Because she could not log in, she was unable to access her work environment.

**Business Impact:**  
User is the lead developer for the payment processing module and needed access before the upcoming sprint deadline.

---

## Initial Assessment

Based on the error message and the user's extended absence, a password expiration issue was suspected.

---

## Troubleshooting Steps

1. Reviewed company Password & Security documentation.
2. Located the user's account in Active Directory.
3. Verified the user's identity using the company verification procedure.
4. Checked account status under Authentication settings.
5. Confirmed the account showed **Password Expired = Yes**.
6. Sent a verification code to the user's registered device.
7. Successfully verified the user's identity.
8. Reset the user's password.
9. Provided a temporary password to the user.
10. Asked the user to log in and create a new password.

---

## Resolution

The user's password had expired due to an extended period of inactivity. After verifying identity, the password was reset and a temporary password was issued. The user successfully logged in and confirmed that access was restored.
